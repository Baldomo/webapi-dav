Negoziazione contenuti

La API risponderà ai seguenti valori di "Accept:" con contenuti appropriati

  • text/plain - Il contenuto sarà restituito come semplice testo.
  • application/json - Il contenuto sarà restituito come oggetto/array JSON

    { message: 'message', subtitle: 'subtitle' }

  • application/xml - Il contenuto sarà restituito come oggetto/array XML <object> <message>message</message> <subtitle>subtitle</subtitle> </object>
